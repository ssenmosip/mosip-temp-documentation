Biometric-APIs

This section details about the service APIs in the Biometric types and biometric attributes modules

  • Biometric Types API

  • Biometrics Attributes API

Biometric types Master API

  • POST /biometrictypes
  • GET /biometrictypes
  • GET /biometrictypes/{id}/{languagecode}

POST /biometrictypes

Master data is required across the platform.

This service will create the list of Biometric types which are used in the MOSIP platform.

Resource URL

POST /biometrictypes

Resource details

Resource DetailsDescription
Response formatJSON
Requires AuthenticationYes

Parameters

NameRequiredDescriptionDefault ValueExample
biometrictypeYesArray of the biometric types

Example Request


{
  "id": "string",
  "version": "string",
  "metadata": {},
  "requesttime": "yyyy-MM-dd'T'HH:mm:ss.SSS'Z'",
  "request": {
      "biometrictypes": [
                {"biometrictype":"string"},
                {"biometrictype":"string"}
                {"languagecode":"string"}
                       ]
                 }
}

Example Response

{
  "id": "string",
  "version": "string",
  "metadata": {},
  "responsetime": "yyyy-MM-dd'T'HH:mm:ss.SSS'Z'",
  "errors": null,
  "response": {
    "biometrictypes": [
                 {"biometrictype":"string"},
                 {"biometrictype":"string"}
                 {"languagecode":"string"}
                     ]
             }
}

Response codes

202

Description: Accepted

400

Description: Bad request

401

Description: Unauthorized

403

Description: Forbidden
